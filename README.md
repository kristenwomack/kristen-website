# Kristen Womack Portfolio Website

Hello! Welcome. This is my personal website. It showcases my writing, featured work, talks, papers, and media mentions.

## Structure

- **Hugo Site**: Static site generator for fast, modern web development
- **Content Organization**: Structured data files for easy content management
- **Responsive Design**: Clean, professional layout optimized for all devices

## Key Sections

- **About**: Professional background and bio
- **Writing**: Published articles and blog posts across various topics
- **Featured Work**: Media coverage, interviews, and industry recognition
- **Talks & Papers**: Speaking engagements and published research

## Development

```bash
# Start local development server
cd site
hugo server

# Build for production
hugo
```

## Deployment

Configured for deployment to GitHub Pages or Azure Static Web Apps.
